# vacstrack
Django BE Software Engineering  CAPASTONE PROJECT

Simple Django app to track vaccine/immunization logs.

Quick start:
- python -m venv .venv
- source .venv/bin/activate
- pip install -r requirements.txt
- python manage.py migrate
- python manage.py createsuperuser
- python manage.py runserver
- Visit http://127.0.0.1:8000/ and login.

Features implemented:
- User authentication (login, logout)
- Vaccine CRUD (create, read, update, delete)
- Dashboard with upcoming reminders (next 30 days)
- CSV export of user data

Submission checklist included in project root for the capstone.
